Main Description

There are a number of ready-to-use templates that present documents and artifacts. Rational Unified Process (RUP) provides templates for use with Microsoft Word, Adobe FrameMaker, HTML, and Microsoft Project. 

RUP also provides guidance on how to use standard templates provided with Rational Rose and Rational SoDA. The Microsoft Word templates can be used with Rational RequisitePro.
