Example: Course Registration System

The Course Registration System Project is a fictional project to develop an on-line course registration system for Wylie College. The project is intended to replace the existing course registration system with a state-of-the-art on-line system that allows student and professor access through PC clients.

Open the Course Registration System Website

The Course Registration System Project example is part of the Wylie College Organization Web example, which shows how an organization can create two complementary Web sites, one for the organization's implementation of the Rational Unified Process and one for a specific project using that implementation.
