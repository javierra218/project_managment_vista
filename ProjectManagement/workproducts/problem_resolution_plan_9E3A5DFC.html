Artifact: Problem_Resolution_Plan
Purpose
The purpose of the Problem Resolution Plan is to describe an orderly process for reporting, analyzing, and resolving problems that occur in the project.

Description
Main DescriptionThe Problem Resolution Plan describes the process used to report, analyze, and resolve problems that occur during the project.
Brief OutlineThis artifact is developed during the Inception phase. Scheduled updates occur based on the results of each Iteration Acceptance Review and Lifecycle Milestone Review. Updates should also occur when changes to problem resolution procedures are identified through quality assurance reviews.
Selected RepresentationThe Project Manager is responsible for maintaining the Problem Resolution Plan.

Tailoring
Impact of not havingThe Problem Resolution Plan may physically be part of the Software Development Plan when the project environment is simple. If the project's interactions are complex (reviews, audits, assessments, and so on) with many stakeholders, it makes sense to separate this.
