Artifact: Iteration_Assessment
Purpose
Each iteration is concluded by an Iteration Assessment,   where the development organization pauses to reflect on what has happened, what was achieved or not and why, and the lessons learned.

Description
Main Description The Iteration Assessment captures the result of an iteration, the degree to which the evaluation criteria were met, lessons learned, and changes to be done.
Notation Iteration Assessments are created at the end of each iteration. They are not updated.
